# Sierra Leone Regional Soil Classification Tool

This web-based platform classifies soils using USCS and AASHTO systems and predicts engineering parameters (CBR, Cu, φ) using regional correlations for Sierra Leone.

## Features
- USCS & AASHTO classification
- Lateritic soil identification
- Regional CBR prediction
- Cu and φ estimation
- Confidence scoring

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
